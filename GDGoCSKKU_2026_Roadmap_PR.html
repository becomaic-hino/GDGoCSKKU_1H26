Roadmap Data Analysis

데이터로 보는 2026년 커리큘럼의 구조와 방향성

기술 분야별 비중

전체 스터디 및 활동 주제를 분석하여 4가지 핵심 도메인으로 분류했습니다.

Insight: Cloud Infra와 App Development가 균형을 이루며, AI 기술이 이를 뒷받침하는 실무 중심의 구조입니다.

월별 활동 강도 (Monthly Pulse)

월별로 예정된 주요 이벤트와 스터디 세션의 수를 집계했습니다.

Insight: 3월(모집/기초), 5월(심화), 11월(고급)에 활동이 집중되며, 시험 기간(4월, 10월)은 조절됩니다.

기술 난이도 성장 곡선

시간 경과에 따른 학습 주제의 기술적 복잡도(Complexity Level) 추이입니다.

Phase 1 (Basic): Workspace, AI Studio 등 진입 장벽이 낮은 도구로 시작
Phase 2 (Core): Docker, Cloud Run, Flutter 등 실무 필수 기술 습득
Phase 3 (Expert): K8s, Terraform 등 엔터프라이즈급 인프라 관리

Quarterly Journey

분기별 핵심 목표와 상세 커리큘럼

Q1. Seeds (기반 구축)

3월 - 기술 입문

Start
  • Recruiting & OT:
    신입부원 모집 및 커뮤니티 온보딩
  • AI & Tools:
    Workspace, AI Studio, Gemini API & CLI
  • Modern Python:
    uv (패키지 매니저), Pydantic, FastAPI 입문
  • Events:
    글로벌 IT 전문가 특강, Hands-on Labs
1
"AI와 대화하는 법을 배우고, 최신 파이썬 도구로 개발 환경을 세팅합니다."
"내 컴퓨터를 벗어나 클라우드 서버에 서비스를 배포하고 앱을 만듭니다."
2

Q2. Roots (심화 학습)

4월 ~ 6월 - 실무 배포

Grow
  • App Development:
    FastAPI + SQLModel, Flutter & Firebase
  • Cloud Infrastructure:
    VPC (Network), Compute Engine (VM), Load Balancing
  • Deployment:
    Docker 컨테이너화, Cloud Run 서버리스 배포
  • Summer Code Snack:
    자체 서비스(Self-hostable App) 개발 및 배포 실전

Q3. Stems (확장)

7월 ~ 9월 - 2학기 준비

Expand
  • Global & Certification:
    S-Global 해외탐방, GCP 자격증(Associate Cloud Engineer) 대비
  • Web & Design:
    Figma (Material 3), Firebase Hosting, Firebase Auth
  • Advanced AI:
    Vertex AI 엔터프라이즈 플랫폼 & Gemini API 심화
3
"자격증으로 실력을 검증하고, 기업용 AI 기술로 스펙트럼을 확장합니다."
"대규모 시스템 관리 기술을 익히고, 1년의 여정을 해커톤으로 마무리합니다."
4

Q4. Harvest (성과)

10월 ~ 12월 - 마무리

Finish
  • Serverless Backend:
    Firebase Cloud Functions, Realtime Database & Storage
  • Enterprise Orchestration:
    Kubernetes (GKE) 대규모 컨테이너 관리
  • IaC (Infrastructure as Code):
    Terraform 인프라 자동화
  • Winter Hackathon:
    연말 연합 해커톤 및 최종 성과 공유
