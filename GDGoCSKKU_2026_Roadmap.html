2026 Season Journey

분기별 핵심 목표와 주요 기술 스택을 확인하세요.

Q1. 씨앗 뿌리기

3월 - 기반 구축 및 기술 입문

  • ●Onboarding: 신입 모집 & OT, Workspace 협업
  • ●AI Basics: Gemini API, AI Studio 프롬프트 엔지니어링
  • ●Modern Python: uv 패키지 매니저, Pydantic, FastAPI 입문
1

"개발 환경을 세팅하고 AI와 대화하는 법을 배웁니다. 솔루션 챌린지 팀 빌딩도 이때 시작됩니다."

Q2. 뿌리 내리기

4월~6월 - 심화 학습 및 실무 배포

  • ●App Dev: Flutter & Firebase, SQLModel DB 연동
  • ●Cloud Core: VPC 네트워킹, Compute Engine, Load Balancing
  • ●Deploy: Docker 컨테이너화 & Cloud Run 서버리스 배포
  • ★Summer Code Snack: 자체 서비스 배포 실전
2

"내 컴퓨터 밖으로 나아갑니다. 클라우드 서버를 직접 빌리고, 앱을 만들어 배포하는 경험을 합니다."

Q3. 줄기 뻗기

7월~9월 - 확장 및 2학기 준비

  • ●Expansion: 2학기 모집, 해외 탐방(S-Global)
  • ●Design & Web: Figma (Material 3), Firebase Hosting
  • ●Advanced AI: Vertex AI 엔터프라이즈 학습
  • ★Autumn Code Snack: GCP 자격증 모의고사
3

"방학 동안 실력을 검증합니다. 자격증 취득을 준비하고, 더 깊은 AI 기술을 탐구합니다."

Q4. 열매 맺기

10월~12월 - 성과 및 마무리

  • ●Serverless: Firebase Cloud Functions & DB
  • ●Orchestration: Kubernetes (GKE) 대규모 관리
  • ●IaC: Terraform 인프라 자동화
  • ★Winter Hackathon: 연말 연합 해커톤 & 성과 공유
4

"엔지니어링의 정점인 자동화와 대규모 관리를 배웁니다. 1년의 노력을 해커톤으로 증명합니다."

Curriculum Deep Dive

데이터로 보는 2026년 활동 분석

스터디 주제별 비중

전체 커리큘럼에서 각 기술 분야가 차지하는 비중입니다. 클라우드 인프라와 앱 개발이 균형을 이룹니다.

분기별 활동 강도

분기별 내부 스터디와 외부 행사(특강/해커톤)의 빈도입니다. Q2와 Q4에 기술 스터디가 집중됩니다.

기술 난이도 성장 곡선

시간이 지남에 따라 다루는 기술의 심도가 어떻게 깊어지는지 보여줍니다. 기초 도구에서 시작해 엔터프라이즈급 자동화 기술로 나아갑니다.
